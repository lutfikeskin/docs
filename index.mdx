## Get Started in 4 Steps

<Steps>
  <Step title="Install and run the app">
    Install dependencies and start the Streamlit application on your machine or server. See the [Quickstart](/quickstart) for the full setup flow.
  </Step>
  <Step title="Add your Gemini API key">
    Open **Settings** from the sidebar and paste your Google Gemini API key. This powers the extraction and natural-language query features.
  </Step>
  <Step title="Upload your first policy PDF">
    Go to **Process Policies**, drag and drop a declarations page PDF, review the extracted data, and save it to your database.
  </Step>
  <Step title="Generate a Certificate of Insurance">
    Head to **Create COI**, search for the saved policy, select a certificate holder, and download your ACORD 25 PDF.
  </Step>
</Steps>

<Note>
  You need a Google Gemini API key to use extraction and natural-language query features. All other features — search, database management, COI generation, and renewals — work without it.
</Note>
